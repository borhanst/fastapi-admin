# Global Search / Command Palette Implementation Plan

## Goal
Add a global search command palette to the topbar search bar, allowing users to search across all registered models and their fields, with results linking to the relevant model list view or filtered list view.

## Current State
- **Topbar search bar** (`fastapi_console/templates/partials/topbar.html:26`) dispatches `open-command-palette` Alpine.js event — but no handler exists
- **No command palette** component is implemented anywhere
- User provided backend API code for search suggestions (pure metadata scan, no DB queries)

## Implementation

### 1. Backend API Endpoint

**File: `fastapi_console/api/search.py`** (NEW)

Create the search suggestions endpoint. Uses `get_current_admin_user` (session cookie auth, not JWT) since this is called from the HTML admin UI.

```python
@router.get("/search/suggestions")
async def get_search_suggestions(request, q):
    await get_current_admin_user(request)  # cookie-based auth
    registry = request.app.state.admin_registry
    # scan registry.all() for model + field matches
    # return {"suggestions": [...], "query": q}
```

Key differences from user's provided code:
- Use `get_current_admin_user` from `fastapi_console.auth.dependencies` (session cookie) instead of `get_api_current_user` (JWT bearer)
- Mounted under `/admin/api/search/suggestions` (within admin prefix, not `/api/`)

### 2. Register the API Router

**File: `fastapi_console/admin/admin_router.py`**

Add the search API router to `_build_router()`:
```python
from fastapi_console.api.search import router as search_api_router
app.include_router(search_api_router, prefix=self.admin_path)
```

### 3. Command Palette Alpine.js Component

**File: `fastapi_console/static/js/admin.js`**

Add `commandPalette` Alpine.js data component inside `alpine:init`:
- `open: false` — controls visibility
- `query: ''` — search input
- `results: []` — suggestions from API
- `selectedIndex: 0` — keyboard navigation index
- `_debounce: null` — debounce timer
- `search()` — fetches `/admin/api/search/suggestions?q=...` with 200ms debounce
- `navigate(url)` — redirects on selection
- Keyboard handlers: ArrowUp/Down to navigate, Enter to select, Escape to close
- Listens for `open-command-palette` event to open
- Listens for Cmd+K / Ctrl+K global shortcut to open

### 4. Command Palette Template

**File: `fastapi_console/templates/partials/command_palette.html`** (NEW)

Modal overlay with:
- Search input (autofocused when opened)
- Results list grouped by type (models first, then fields)
- Each result shows icon + label + sublabel
- Keyboard navigation highlighting
- Click outside / Escape to close

### 5. Include in Base Layout

**File: `fastapi_console/templates/base.html`**

Add after the confirm dialog template:
```html

```
