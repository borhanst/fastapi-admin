# Plan: Permission-Based Sidebar Rendering

## Goal

Sidebar menu items (model links, custom nav items, Dashboard, Settings) are hidden when the current user lacks the required permission. Users with no role or no matching `AdminPermission` row see only items without a permission requirement.

## Design Decisions

- **No new DB tables**. `AdminPermission.table_name` is already a free-form `String(255)` ã custom permission names (e.g. `"view_reports"`) work as table_name values.
- **No new seed data**. Custom permissions are added by users via `SeedRole(permissions={"perm_name": {"view": True}})`.
- **Backward compatible**. Every new field defaults to `None`, meaning "visible to everyone". Existing configs need zero changes.
- **`NavItemConfig.permission`** already maps to `BuiltNavItem.permission_table` which is looked up in `permissions_map`. This works for model-linked items already. The gap is that custom nav items can only reference model table names. No change needed for this ã `AdminPermission.table_name` is free-form.

**build_sidebar_context()** (~line 49-113) ã three changes:

a) After building `permissions_map`, compute visibility flags:
```python
dashboard_visible = (
    is_superuser
    or self.dashboard_permission is None
    or (permissions_map.get(self.dashboard_permission) and permissions_map[self.dashboard_permission].can_view)
)
settings_visible = (
    is_superuser
    or self.settings_permission is None
    or (permissions_map.get(self.settings_permission) and permissions_map[self.settings_permission].can_view)
)
```

b) Add group-level permission check (before item filter loop):
```python
if not (group.permission_table is None
        or is_superuser
        or (permissions_map.get(group.permission_table) and permissions_map[group.permission_table].can_view)):
    continue
```

c) Filter children recursively in the item filter loop. Extract a helper or add inline recursion that filters `item.children` using the same logic.

d) Return new flags in context dict:
```python
return {
    "nav_groups": filtered_groups,
    "permissions_map": permissions_map,
    "current_user": user,
    "dashboard_visible": dashboard_visible,
    "settings_visible": settings_visible,
}
```
